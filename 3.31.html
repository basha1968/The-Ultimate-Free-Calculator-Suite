Formula Syntax Guide

Elements: Use standard symbols (H, He, C, Na, etc.). First letter uppercase, second lowercase.
Numbers: Use after elements to indicate quantity (H2O). Omit for single atoms (NaCl).
Parentheses: Group elements (e.g., (NH4)2SO4). Supports nesting.
Spaces: Not required. Formula is case-sensitive.
